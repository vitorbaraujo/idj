## Execução do jogo

* __Para executar o jogo, na pasta raíz do projeto, siga os seguintes passos:__

* __Limpe quaisquer resquícios do execuções anteriores__

`make dist-clean`

* __Compile os arquivos do jogo__

`make`

* __Por fim, execute o jogo__

`make run`
